09
Crear TodoItem.jsx

Dentro de src/components/ crea TodoItem.jsx. Este componente muestra una tarea individual y recibe texto, completada y onToggle como props.

function TodoItem({ texto, completada, onToggle }) {
  return (
    <div className="flex items-center gap-3 p-3 border rounded-lg mb-2">
      <input
        type="checkbox"
        checked={completada}
        onChange={onToggle}
        className="w-4 h-4"
      />
      <span className={completada ? 'line-through text-gray-400' : ''}>
        {texto}
      </span>
    </div>
  );
}

export default TodoItem;
La clase de Tailwind se aplica condicionalmente con un ternario: si completada es true aplica line-through text-gray-400, si no, no aplica ninguna clase extra.
10
Crear TodoList.jsx

Este componente recibe el array de tareas y renderiza un TodoItem por cada una.

import TodoItem from './TodoItem';

function TodoList({ tareas, onToggle }) {
  return (
    <div>
      {tareas.map((tarea) => (
        <TodoItem
          key={tarea.id}
          texto={tarea.texto}
          completada={tarea.completada}
          onToggle={() => onToggle(tarea.id)}
        />
      ))}
    </div>
  );
}

export default TodoList;
onToggle={() => onToggle(tarea.id)} — se envuelve en una arrow function para poder pasar el id de la tarea al handler del padre.
11
Crear TodoForm.jsx

El estado del input vive aquí porque solo lo necesita este componente. Implementa el formulario con validación básica.

import { useState } from 'react';

function TodoForm({ onAgregar }) {

  const [texto, setTexto] = useState('');

  function handleSubmit(e) {
    e.preventDefault();         // evita que la página se recargue
    if (!texto.trim()) return;  // no agregar si el input está vacío
    onAgregar(texto);
    setTexto('');               // limpiar el input después de agregar
  }

  return (
    <form onSubmit={handleSubmit} className="flex gap-2 mb-6">
      <input
        type="text"
        value={texto}
        onChange={(e) => setTexto(e.target.value)}
        placeholder="Nueva tarea..."
        className="flex-1 border rounded-lg px-3 py-2"
      />
      <button
        type="submit"
        className="bg-blue-500 text-white px-4 py-2 rounded-lg"
      >
        Agregar
      </button>
    </form>
  );
}

export default TodoForm;
⚠️ e.preventDefault() es necesario porque por defecto un formulario HTML recarga la página al hacer submit. En React siempre hay que evitar ese comportamiento.
⚠️ texto.trim() elimina espacios en blanco al inicio y al final — evita agregar tareas vacías o con solo espacios.
12
Configurar App.jsx con el estado global

El estado de las tareas vive en App.jsx porque lo necesitan varios componentes. Importa los componentes y define el estado inicial.

TodoForm espera onAgregar (paso 11) y TodoList espera onToggle (paso 10). Por ahora usa handlers vacíos; en el paso 13 conectarás agregarTarea y en el paso 14 toggleTarea. 
import { useState } from 'react';
import TodoForm from './components/TodoForm';
import TodoList from './components/TodoList';

function App() {

  const [tareas, setTareas] = useState([
    { id: 1, texto: 'Aprender React', completada: false },
    { id: 2, texto: 'Hacer la practica', completada: false },
  ]);

  return (
    <div className="max-w-md mx-auto mt-10 p-6">
      <h1 className="text-2xl font-bold mb-6">Mi lista de tareas</h1>
      <TodoForm onAgregar={() => {}} />
      <TodoList tareas={tareas} onToggle={() => {}} />
    </div>
  );
}

export default App;
13
Agregar la función para agregar tareas

Crea agregarTarea en App y pásala como prop a TodoForm.

Sustituye el onAgregar={() => {}} del paso 12 por onAgregar={agregarTarea}. 
function agregarTarea(texto) {
  const nuevaTarea = {
    id: Date.now(),
    texto,
    completada: false,
  };
  setTareas([...tareas, nuevaTarea]);
}

// Actualizar el return:
<TodoForm onAgregar={agregarTarea} />
Date.now() devuelve el número de milisegundos desde 1970 — forma rápida de generar ids únicos sin librerías externas.
[...tareas, nuevaTarea] crea un array nuevo con todas las tareas anteriores más la nueva. Nunca usar tareas.push() porque muta el array original y React no detecta el cambio.
14
Agregar la función para completar tareas

Crea toggleTarea en App y pásala hasta TodoItem a través de TodoList.

Sustituye el onToggle={() => {}} del paso 12 por onToggle={toggleTarea}. 
function toggleTarea(id) {
  setTareas(tareas.map((tarea) =>
    tarea.id === id ? { ...tarea, completada: !tarea.completada } : tarea
  ));
}

// Actualizar el return:
<TodoList tareas={tareas} onToggle={toggleTarea} />
.map() recorre todas las tareas. Cuando encuentra la que tiene el id buscado, devuelve una copia con completada invertido (!tarea.completada). Las demás las devuelve sin cambio.
🎯 Reto extra
Botón de eliminar

Agrega un botón ✕ en cada TodoItem para eliminar la tarea. Requiere crear eliminarTarea en App.jsx y pasarla hasta el componente.

App.jsx — nueva función y prop en la lista:
function eliminarTarea(id) {
  setTareas(tareas.filter((tarea) => tarea.id !== id));
}

// En el return, pasa también onEliminar:
<TodoList
  tareas={tareas}
  onToggle={toggleTarea}
  onEliminar={eliminarTarea}
/>
TodoList.jsx — recibir la prop y pasarla al ítem:
function TodoList({ tareas, onToggle, onEliminar }) {
  return (
    <div>
      {tareas.map((tarea) => (
        <TodoItem
          key={tarea.id}
          texto={tarea.texto}
          completada={tarea.completada}
          onToggle={() => onToggle(tarea.id)}
          onEliminar={() => onEliminar(tarea.id)}
        />
      ))}
    </div>
  );
}
TodoItem.jsx — recibir la prop y agregar el botón:
function TodoItem({ texto, completada, onToggle, onEliminar }) {
  return (
    <div className="flex items-center gap-3 p-3 border rounded-lg mb-2">
      <input
        type="checkbox"
        checked={completada}
        onChange={onToggle}
        className="w-4 h-4"
      />
      <span className={completada ? 'line-through text-gray-400' : ''}>
        {texto}
      </span>
      <button
        type="button"
        onClick={onEliminar}
        className="ml-auto text-red-400 hover:text-red-600"
        aria-label="Eliminar tarea"
      >
        ✕
      </button>
    </div>
  );
}
.filter() devuelve un nuevo array con todas las tareas excepto la que tiene el id recibido. El original nunca se modifica.
