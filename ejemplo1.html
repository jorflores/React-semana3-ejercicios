Ejemplo 1

🔢 Contador Interactivo

useState manejo de eventos re-render className CSS por componente ⏱ 35–40 min
⚙️ Setup del proyecto
01
Crear el proyecto
npm create vite@latest contador-app -- --template react
02
Entrar al proyecto
cd contador-app
03
Instalar dependencias e iniciar el servidor

La primera vez, instala paquetes dentro de la carpeta del proyecto. Luego levanta el servidor de desarrollo.

npm install
⚠️ Espera a que termine npm install antes de ejecutar el siguiente comando.
npm run dev
04
Limpiar archivos base
  • En src/index.css reemplaza todo por un reset básico
  • En src/App.css borra todo el contenido
  • En src/App.jsx deja solo el componente vacío
src/index.css
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

body {
  font-family: sans-serif;
}
src/App.jsx
function App() {
  return (
    <div>
      <h1>Contador App</h1>
    </div>
  );
}

export default App;
05
Crear la carpeta de componentes

Dentro de src/, crea la carpeta components/.

📋 Pasos del ejercicio
06
Crear el archivo del componente

Dentro de src/components/ crea el archivo Contador.jsx.
Si usas VS Code con snippets (p. ej. ES7+), puedes escribir rfce + Tab para generar la estructura base; si no, crea una función Contador que devuelva un <div> vacío y ajusta el nombre del componente.

07
Importar useState

En la primera línea, importa el hook que necesitas para manejar estado.

import { useState } from 'react';
08
Declarar el estado

Dentro de la función, antes del return, declara la variable de estado con valor inicial 0.

const [contador, setContador] = useState(0);
useState(0) → el 0 es el valor inicial. La tupla da dos cosas: el valor actual (contador) y la función para actualizarlo (setContador).
09
Crear las funciones

Crea tres funciones: una para incrementar, una para decrementar y una para reiniciar el contador.

function incrementar() {
  setContador(contador + 1);
}

function decrementar() {
  setContador(contador - 1);
}

function reiniciar() {
  setContador(0);
}
⚠️ El valor del estado solo cambia con el setter (setContador). No reasignes contador ni intentes mutarlo: React no lo trata como una actualización y la UI no reflejaría el cambio.
También puedes usar la forma funcional: setContador((c) => c + 1). React pasa el valor más reciente del estado a esa función, así que si llamas al setter varias veces seguidas en el mismo evento, cada llamada parte del valor actualizado anterior (con setContador(contador + 1) dos veces seguidas podrías quedarte solo con +1).
10
Escribir el JSX

En el return, construye la UI: contenedor, título, número y tres botones (−, Reset, +) con sus manejadores de eventos.

return (
  <div className="contador-contenedor">
    <h2 className="contador-titulo">Contador Interactivo</h2>
    <span className="contador-numero">{contador}</span>
    <div className="contador-botones">
      <button className="btn btn-restar" onClick={decrementar}>−</button>
      <button className="btn btn-reset"  onClick={reiniciar}>Reset</button>
      <button className="btn btn-sumar"  onClick={incrementar}>+</button>
    </div>
  </div>
);
⚠️ En React se usa className en lugar de class porque class es palabra reservada de JavaScript.
⚠️ Escribe onClick={incrementar} sin paréntesis. Con paréntesis — onClick={incrementar()} — el código se ejecuta al cargar, no al hacer click.
11
Exportar el componente

Al final del archivo, exporta el componente como default.

export default Contador;

13
Importar y usar en App.jsx

Importa el componente Contador en App.jsx y úsalo en el JSX.

import Contador from './components/Contador';

function App() {
  return (
    <div>
      <Contador />
    </div>
  );
}

export default App;
