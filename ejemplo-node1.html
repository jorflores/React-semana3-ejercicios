Ejemplo Node 1

API REST con Express

Node.js Express ES Modules REST CRUD JSON 45-50 min
Objetivo
00
Construir una API REST de usuarios

En este ejercicio crearás un servidor con Express que expone rutas HTTP para administrar usuarios.

Como todavía no hay base de datos, los registros vivirán en un arreglo dentro del archivo server.js. Al reiniciar el servidor, los cambios se perderán.

Una API REST expone recursos a través de rutas HTTP. Cada método representa una operación CRUD sobre los datos.
Rutas que debe tener la API
GET
/usuarios
Obtener lista de usuarios
GET
/usuarios/:id
Obtener un usuario específico
POST
/usuarios
Crear un nuevo usuario
PUT
/usuarios/:id
Actualizar un usuario
DELETE
/usuarios/:id
Eliminar un usuario
Preparación del proyecto
01
Crear carpeta e inicializar Node.js

Abre una terminal y crea un proyecto nuevo para la API.

Terminal
mkdir api-usuarios
cd api-usuarios
npm init -y
02
Instalar Express

Express nos permite crear rutas HTTP de forma sencilla.

Terminal
npm install express
03
Configurar módulos y script de inicio

Abre package.json, agrega "type": "module" y cambia la sección scripts.

Con esta configuración podrás importar librerías usando import en lugar de require.

package.json
{
  "type": "module",
  "scripts": {
    "start": "node server.js"
  }
}
Código base
04
Crear el archivo server.js

Crea un archivo llamado server.js y agrega el servidor básico con un arreglo de usuarios.

server.js
import express from 'express';

const app = express();
const PORT = 3000;

app.use(express.json());

let usuarios = [
  { id: 1, nombre: 'Ana', correo: 'ana@tec.mx' },
  { id: 2, nombre: 'Luis', correo: 'luis@tec.mx' },
  { id: 3, nombre: 'Marta', correo: 'marta@tec.mx' }
];

app.get('/', (req, res) => {
  res.json({ mensaje: 'API REST de usuarios con Express' });
});

app.listen(PORT, () => {
  console.log(`Servidor escuchando en http://localhost:${PORT}`);
});
La línea import express from 'express' funciona porque agregaste "type": "module" en package.json. El middleware express.json() permite leer datos JSON enviados en el cuerpo de las peticiones POST y PUT.
Implementación de rutas CRUD
05
GET /usuarios - listar usuarios

Devuelve el arreglo completo de usuarios.

server.js
app.get('/usuarios', (req, res) => {
  res.json(usuarios);
});
06
GET /usuarios/:id - buscar por id

Recibe el id desde la URL y busca un usuario dentro del arreglo.

server.js
app.get('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const usuario = usuarios.find((item) => item.id === id);

  if (!usuario) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  res.json(usuario);
});
07
POST /usuarios - crear usuario

Lee nombre y correo desde el cuerpo de la petición y agrega un nuevo objeto al arreglo.

server.js
app.post('/usuarios', (req, res) => {
  const { nombre, correo } = req.body;

  if (!nombre || !correo) {
    return res.status(400).json({ mensaje: 'Nombre y correo son obligatorios' });
  }

  const nuevoUsuario = {
    id: usuarios.length + 1,
    nombre,
    correo
  };

  usuarios.push(nuevoUsuario);
  res.status(201).json(nuevoUsuario);
});
08
PUT /usuarios/:id - actualizar usuario

Actualiza los datos de un usuario existente. Si un campo no viene en la petición, conserva su valor anterior.

server.js
app.put('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const usuario = usuarios.find((item) => item.id === id);

  if (!usuario) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  const { nombre, correo } = req.body;

  usuario.nombre = nombre || usuario.nombre;
  usuario.correo = correo || usuario.correo;

  res.json(usuario);
});
09
DELETE /usuarios/:id - eliminar usuario

Elimina un usuario del arreglo usando su id.

server.js
app.delete('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const existe = usuarios.some((item) => item.id === id);

  if (!existe) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  usuarios = usuarios.filter((item) => item.id !== id);
  res.json({ mensaje: 'Usuario eliminado correctamente' });
});
Archivo completo
10
Código final de server.js

Tu archivo completo debe quedar así:

server.js
import express from 'express';

const app = express();
const PORT = 3000;

app.use(express.json());

let usuarios = [
  { id: 1, nombre: 'Ana', correo: 'ana@tec.mx' },
  { id: 2, nombre: 'Luis', correo: 'luis@tec.mx' },
  { id: 3, nombre: 'Marta', correo: 'marta@tec.mx' }
];

app.get('/', (req, res) => {
  res.json({ mensaje: 'API REST de usuarios con Express' });
});

app.get('/usuarios', (req, res) => {
  res.json(usuarios);
});

app.get('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const usuario = usuarios.find((item) => item.id === id);

  if (!usuario) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  res.json(usuario);
});

app.post('/usuarios', (req, res) => {
  const { nombre, correo } = req.body;

  if (!nombre || !correo) {
    return res.status(400).json({ mensaje: 'Nombre y correo son obligatorios' });
  }

  const nuevoUsuario = {
    id: usuarios.length + 1,
    nombre,
    correo
  };

  usuarios.push(nuevoUsuario);
  res.status(201).json(nuevoUsuario);
});

app.put('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const usuario = usuarios.find((item) => item.id === id);

  if (!usuario) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  const { nombre, correo } = req.body;

  usuario.nombre = nombre || usuario.nombre;
  usuario.correo = correo || usuario.correo;

  res.json(usuario);
});

app.delete('/usuarios/:id', (req, res) => {
  const id = Number(req.params.id);
  const existe = usuarios.some((item) => item.id === id);

  if (!existe) {
    return res.status(404).json({ mensaje: 'Usuario no encontrado' });
  }

  usuarios = usuarios.filter((item) => item.id !== id);
  res.json({ mensaje: 'Usuario eliminado correctamente' });
});

app.listen(PORT, () => {
  console.log(`Servidor escuchando en http://localhost:${PORT}`);
});
Probar la API
11
Ejecutar el servidor

Inicia el proyecto con el script configurado.

Terminal
npm start

Después abre http://localhost:3000/usuarios en el navegador para ver la lista inicial.
