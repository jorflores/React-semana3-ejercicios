👤 Buscador de Usuarios

React Router useEffect fetch useParams Link ⏱ Entrega en 1 semana Individual
📋 Descripción

Construye una app de dos páginas que consuma la API pública JSONPlaceholder para mostrar información de usuarios. La app permite buscar un usuario por ID y ver su perfil completo en una página de detalle.

La página principal tiene un selector o input donde el usuario elige un ID del 1 al 10. Al hacer clic en "Buscar", navega a la ruta /usuario/:id donde se hace el fetch y se muestran los datos. No se requieren componentes extra más allá de las dos páginas y un Navbar.

La API no requiere registro ni API key — funciona directamente desde el navegador.

🌐 La API — JSONPlaceholder
URL base:
https://jsonplaceholder.typicode.com/users/:id
Ejemplo de respuesta para /users/1 — solo las propiedades que usarás:
{
  "id": 1,
  "name": "Leanne Graham",
  "email": "Sincere@april.biz",
  "phone": "1-770-736-8031 x56442",
  "website": "hildegard.org",
  "address": {
    "city": "Gwenborough"
  },
  "company": {
    "name": "Romaguera-Crona"
  }
}
La API tiene exactamente 10 usuarios (IDs del 1 al 10). Si se pide un ID fuera de ese rango, la respuesta llega vacía — lo cual es una buena oportunidad para mostrar un mensaje de error.

💡 Pista — useNavigate

Para navegar desde Home al hacer clic en "Buscar", necesitas useNavigate en lugar de Link — porque la navegación depende de lo que el usuario escribió en el input, no de una URL fija.

Se usa así:

import { useNavigate } from 'react-router-dom';

function Home() {
  const [id, setId] = useState(1);
  const navigate = useNavigate();

  function handleBuscar() {
    navigate(`/usuario/${id}`);   // navega a /usuario/3, /usuario/7, etc.
  }

  return (
    <div>
      <input
        type="number"
        min="1"
        max="10"
        value={id}
        onChange={(e) => setId(e.target.value)}
      />
      <button onClick={handleBuscar}>Buscar</button>
    </div>
  );
}
